# HTU Anonymous Complaint System

A modern, secure, and anonymous complaint management system for educational institutions with ultra-modern UI/UX.

## ✨ Features

### 🎨 **Modern UI/UX**
- Glassmorphism design with rich gradients
- Dark/Light theme toggle
- Responsive design for all devices
- Smooth animations and transitions
- Twitter-like complaint feed
- Interactive admin dashboard with charts

### 🔐 **Authentication System**
- Student registration with index number
- Auto-generated institutional emails
- Email verification (OTP/Token)
- Secure session management
- Password reset functionality
- Account verification system

### 📝 **Complaint Management**
- Anonymous complaint submission
- Categorized complaints (Hostel, Campus, etc.)
- File attachments support
- Voting system (upvote/downvote)
- Commenting system
- Status tracking (Pending → Published → Resolved)
- Location tagging

### 👥 **Admin System**
- Multi-level admin roles (Super Admin, Category Admin)
- Category-based complaint assignment
- Interactive DataTables for management
- Real-time notifications
- Comprehensive audit logs
- Export functionality (PDF, Excel)
- User management

### ⚡ **Technical Features**
- PDO database connection (MySQL)
- CSRF protection
- Input validation & sanitization
- File upload security
- AJAX-based interactions
- Gmail SMTP integration
- RESTful API endpoints

## 📋 Prerequisites

- PHP 7.4 or higher
- MySQL 5.7 or higher
- Apache/Nginx web server
- Composer (for dependencies)
- Gmail account (for email services)
